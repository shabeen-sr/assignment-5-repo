Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM.
1. getElementById()
2. getElementsByClassName()
3. querySelector()
4. querySelectorAll()
5. getElementsByTagNam()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. In innerHTML it shows HTMl content includes tags and text.
2. In innerText it shows the text without HTML.
3. In textContent it shows raw text without any HTMl and formating.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where I can use a single event listener to a parent element, If there are many child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a technique where event starts from the target element and triggered the event spread upwards to its parent elements.
